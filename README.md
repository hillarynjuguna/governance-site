# Oscillatory Fields Governance Site

This repository contains the source code for the Oscillatory Fields Governance Site.

## Architecture

This project is built using:
- **Astro**: High-performance static site generation
- **React**: Component architecture
- **Tailwind CSS**: Utility-first styling

### Repository Structure

- `src/pages/`: File-based routing entry points.
- `src/components/`: Reusable React/Astro UI components.
- `src/content/` & `src/data/`: Centralized content schemas and data.
- `src/styles/`: Global styling and Tailwind configurations.
- `public/`: Unprocessed static assets.

## Local Development

```bash
# Install dependencies
npm install

# Start the local development server
npm run dev

# Build for production
npm run build
```
